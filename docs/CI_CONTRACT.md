# Контракт: как задачи превращаются в готовый .exe

## Цель
Пользователь ставит задачу → репозиторий автоматически прогоняет проверки и собирает установщик Windows → результат доступен как готовый файл (.exe).

## Definition of Done (DoD)
Задача считается сделанной, только если:
1) `windows-ci` ✅ зелёный (Lint + Typecheck + Dist)
2) В артефакте `installer-windows` есть установщик `*.exe`
3) (опционально) При релизе по тегу `v*` создан GitHub Release с вложенным `*.exe`

## Где брать готовый файл
- После любого успешного `windows-ci`: Actions → Run → Artifacts → `installer-windows`
- После тега `vX.Y.Z`: Releases → нужный релиз → Assets → `*.exe`
